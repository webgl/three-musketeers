click

click

Source:
  • nodeComponents/click/click.js, line 11

Given a node within the scene, click it's position. You can pass in true for debug to see the cursor position.

Example

$$$
.find('Cube_1') // an element in the scene with a name of `Cube_1`
.click(); // clicks the node

 $$$
.find('Cube_1')
.click(true); // you can also pass in `debug` as `true` to see the cursor
